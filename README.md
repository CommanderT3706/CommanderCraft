# CommanderCraft

## How to install?

Here's how to install the CommanderCraft modpack for you to play in!

### MultiMC

1. Download .zip from Releases
2. Start MultiMC
3. Go to Add instance > Import from zip
4. Select the downloaded file
5. Press 'OK'
6. Enjoy!

### Manual

1. Download and install the lastest version of Minecraft Forge 1.18.2: https://files.minecraftforge.net/net/minecraftforge/forge/
2. Download the respository
3. Extract the downloaded .zip
4. Go to /Mods
5. Copy all the files to *%appdata%/.minecraft/mods/*
6. Start the Minecraft Launcher and launch Forge
7. Enjoy!
